CoCreate-crud-client

An useful CRUD api operate Create, read, update, delete with built in database. Can be used as a firebase alternative. Easily configured using HTML5 attributes and/or JavaScript API.

Install

 
npm i @cocreate/cocreate-crud-client

Or you can use cdn link: 

<script>https://cdn.cocreate.app/crud-client/latest/CoCreate-crud-client.min.js</script>

Usage

crud-client usage content 

<div></div>

Attributes

  • crud-client string optional

    crud-client-attribute

  • crud-client string optional

    crud-client-attribute
